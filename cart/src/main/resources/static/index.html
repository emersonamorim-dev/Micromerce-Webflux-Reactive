Cart Service API Documentation

RESTful API documentation for the reactive cart management service.

POST /api/v1/carts

Create a new shopping cart for a user.

Request Body:

{
    "userId": "string"
}

POST /api/v1/carts/{cartId}/items

Add an item to a cart.

Request Body:

{
    "productId": "string",
    "quantity": "integer",
    "price": "decimal",
    "productName": "string"
}

PUT /api/v1/carts/{cartId}/checkout

Checkout a cart.

GET /api/v1/carts/user/{userId}

Get all carts for a user.

GET /api/v1/carts/{cartId}

Get a specific cart by ID.

DELETE /api/v1/carts/{cartId}/items/{itemId}

Remove an item from a cart.
